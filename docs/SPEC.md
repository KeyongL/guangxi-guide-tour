## 一句话目标

建一个面向欧美自由行游客的广西英文攻略网站（以北海/涠洲岛为大本营，覆盖东兴、崇左、百色、河池），占领英文搜索的信息真空，3个月内产生真实客源咨询。

## 用户故事（谁、做什么、为什么）

1. **主要**：计划或正在越南北部/东南亚旅行的欧美背包客（25–45岁，英语母语/流利，多为48国免签名单或240小时过境免签适用国籍），在 Google / Reddit 搜索 Beihai / Weizhou Island / Detian Waterfall / China border crossing 信息时找到本站，阅读攻略后通过联系入口咨询行程。
2. **次要**：已在中国生活的外国人（南宁高校留学生/外教、珠三角外侨），寻找周末/小长假短途目的地——他们没有签证/支付/网络摩擦，是冷启动期的第一批真实客人。
3. **站长**（用户本人，北海，博士，英语流利，每天约8小时可投入）：用 Markdown 写文章即可更新网站，不需要碰代码。

## 输入 / 输出定义（含具体示例）

- **输入**：站长撰写的英文 Markdown 文章（攻略/行程/实用指南），含标题、slug、摘要、封面图、正文。
  - 示例：`content/guides/visa-free-transit-beihai.md` → 标题 "How to Visit Beihai Visa-Free: China's 240-Hour Transit Policy Explained"
- **输出**：
  - SEO 优化的静态网页（英文），全球 CDN 分发；
  - 咨询邮件：访客提交联系表单 → 站长邮箱收到含姓名/邮箱/留言的邮件。
  - 示例：访客在行程页点击 "Plan my trip" → 填表 → 站长收到邮件，页面显示 "Thanks, I'll reply within 24h."

站点覆盖六个区域，但**过境免签适用性不同，内容必须明确标注**，避免误导读者非法越区：

| 区域 | 核心资源 | 240h过境免签区内？ | 内容标注要求 |
|---|---|---|---|
| 北海 + 涠洲岛 | 银滩、火山岛、老街骑楼 | ✅ 是（且是入境口岸） | 无限制 |
| 东兴/防城港 | 中越双口岸（陆+水）、边贸 | ✅ 是 | 陆路口岸对第三国护照的通行现状**需实地核实后再发布** |
| 河池（巴马） | 长寿乡IP、喀斯特 | ✅ 是 | 无限制 |
| 崇左 | 德天瀑布（中越跨境）、友谊关 | ❌ 否 | 必须标注"仅适用30天免签国籍（欧洲/澳新等48国）或持签证者；过境免签者不可前往" |
| 百色 | 乐业天坑、少数民族文化 | ❌ 否 | 同上 |
| 南宁（枢纽） | 交通中转、美食 | ✅ 是（入境口岸） | 无限制 |

## 核心内容框架：两条流向（用户确认的攻略主线）

攻略按"怎么进 → 怎么走 → 怎么出"的路线流向组织，而非孤立的目的地介绍：

**流向 A —— 飞入广西 → 玩 → 陆路去越南**（旗舰路线）
- 签证逻辑：240小时过境免签要求"前往第三国"——越南就是那个第三国，这条流向天然合法自洽；48国免签国籍同样适用且更自由。
- 路线骨架：飞北海/南宁 → 北海/涠洲岛 → 东兴出境 → 芒街/下龙湾/河内。
- ⚠️ 待核实：过境免签旅客能否从东兴**陆路出境**（政策只明确了机场/邮轮港入境口岸）。若核实不通过，TWOV 版本降级为"原机场飞离"，48国免签版本不受影响。
- ⚠️ 友谊关在崇左（非过境免签区），过境免签旅客不可走友谊关方向，只能东兴。

**流向 B —— 从越南陆路直接进广西 → 玩 → 飞离或返回越南**
- 签证逻辑：过境免签**不支持陆路入境**（入境口岸限机场/邮轮港），此流向仅适用48国免签国籍（欧洲/澳新等，不含美国）或持签证者——文章必须开头就讲清楚。
- 路线1：芒街 → 东兴（陆桥/船渡）→ 防城港/北海海岸线。
- 路线2：河内 → 友谊关/凭祥（或河内—南宁国际列车）→ 南宁（此线经崇左，注意签证分区标注）。

## 验收标准（每条可测试）

**上线验收（V1 完成的定义）：**
1. 当上线时，站点应包含 ≥8 篇英文攻略（首发批）：①流向A旗舰指南 "Fly into Guangxi, leave overland to Vietnam"（240h过境免签实操）②流向B旗舰指南 "Vietnam to China overland"（芒街→东兴 + 河内→南宁两线，含签证适用性说明；东兴段实地核实后发布）③外卡绑微信/支付宝支付指南 ④国际eSIM选购 ⑤涠洲岛完全指南 ⑥北海老街与骑楼 ⑦海鲜市场怎么吃 ⑧流向A版 3–5 天海岸线行程（北海—涠洲—东兴—出境越南）。
2. 上线后第一个月内补齐第二批 ≥3 篇（扩展区域）：⑨德天瀑布+友谊关（崇左，含签证分区提醒）⑩乐业天坑（百色，含签证分区提醒）⑪巴马长寿乡（河池）；并上线流向B版"广西边境 7–10 天大环线"行程页（越南入境→东兴/南宁→崇左→百色→河池，仅48国免签/持签证适用）。
3. 当访客用手机（375px 宽）访问任一页面时，应无横向滚动、正文可读、导航可用。
4. 当访客提交联系表单时，站长邮箱应收到邮件，且页面不刷新即显示成功提示；表单服务不可用时应降级显示 mailto 链接。
5. 当搜索引擎抓取时，每页应有唯一 title、meta description、Open Graph 标签；`/sitemap.xml` 与 `/robots.txt` 可访问。
6. 当用 Lighthouse 移动端测试首页与任一文章页时，Performance ≥ 85、SEO ≥ 90。
7. 当文章涉及崇左/百色时，页面应显著展示签证分区提醒框（模板组件，不靠作者手写记忆）。

**运营目标（上线后3个月，非代码验收，用户确认的成功标准）：**
8. 收到 ≥10 封来自真实外国游客的咨询（邮件/表单）。
9. 目标关键词（"Beihai travel guide" / "Weizhou Island travel" / "Detian Waterfall from Vietnam"）进入 Google 前两页。
10. 月独立访客 ≥ 500（主要来自搜索与 Reddit）。
11. 通过网站来的客人实际成行第一单（哪怕成本价）。

## 范围外（明确不做什么）

- V1 无在线支付、无用户注册、无预订引擎、无后台管理系统（内容即文件）。
- 只做英文，不做中文/多语言版本。
- 不提供签证法律意见——只做政策事实陈述并链接官方来源（国家移民管理局等），并标注"政策可能变化，以官方为准"。
- **不发布任何翻墙/VPN 教程类内容**（站长身处境内，合规红线）；网络建议仅限国际漫游 eSIM 的事实性介绍。
- 不做小程序 / App / 不接社交平台自动发布。
- 桂林/阳朔不做深度内容（英文信息已饱和、竞争无优势），最多在行程页作为衔接点提及。

## 技术栈决策

- 语言与运行时：TypeScript（Node.js 构建期）+ Markdown 内容
- 后端框架：无（纯静态站点）
- 前端框架：Astro（静态站点生成器）+ 原生 CSS，不引入重前端框架
- 数据库：无（内容为 Git 管理的 Markdown 文件）
- 部署目标：Cloudflare Pages（免费、全球 CDN；受众在境外、站点托管在境外域名下，无需 ICP 备案；自定义域名需站长购买，约 $10/年）
- 理由：受众全部在境外，静态站 SEO 最优、托管零成本、无服务器可维护性负担；站长后续只需写 Markdown 提交即可更新。

## 交互增强与城市名片（2026-07-03 /change 新增，排期 V1.1）

> 定位：**流向攻略仍是 SEO 与叙事主线；地图与城市名片是挂在行程页上的"参考层"，不改变"按流向组织、不按孤立景点"的核心框架。** 均排在 V1 上线之后作为快速跟进，避免拖慢"3 个月出真实客源"。

**交互路线地图（Leaflet + OpenStreetMap）**
- 行程页展示流向 A/B 的路线折线 + 可点击城市标记；点击标记弹出该城市名片。
- 渐进增强红线：SEO 依赖的路线/城市信息必须以静态 HTML 渲染（Astro 构建期由 frontmatter 生成），地图叠加其上；关闭 JS 或地图加载失败时降级为静态占位图 + 文字路线，页面仍完整可读。
- 性能：Leaflet 与瓦片懒加载、仅在含地图的页面加载，守住 Lighthouse 移动端 Performance ≥ 85。
- 选 OSM 而非 Google 的理由：免密钥免账单、站长身处境内预览/写作更顺（Google 地图境内不稳）、受众在境外加载正常。

**城市名片数据模型（存于文章/城市 Markdown frontmatter，站长只填字段、不碰代码）**
| 字段 | 说明 |
|---|---|
| `name_en` / `name_local` | 英文名 + 本地名 |
| `region` + `visa_zone` | 区域 + 签证分区；**复用现有签证徽章组件**，崇左/百色自动挂"非过境免签区"警告（验收标准 7 集中在此，不靠作者记忆） |
| `days` | 推荐游玩天数 |
| `daily_budget` + `budget_verified_on` | 日预算区间（¥ + ~$）+ **最后核实日期**（YYYY-MM-DD）；价格会变，无核实日期的预算数据视为过期，宁缺毋滥 |
| `must_eat` | 2–3 道必吃（含大致价格） |
| `attractions` | 2–3 个招牌景点 |
| `activity_level` + `adventure_hook` | 活动强度评级（Chill ↔ Adventurous）+ 一个招牌硬核玩法（如涠洲岛=自由潜/火山海岸徒步；德天=竹筏冲到瀑布脚下） |
| `flow_links` | 指向经过本城市的流向攻略链接 |

**"冒险"内容的合规护栏（硬约束）**
- 面向的是"25–45 岁欧美自由行背包客偏好冒险"这一**人群特征**，不作任何肤色/种族层面的表述。
- "冒险"仅限合法、安全的户外/体验玩法；**严禁**引导走非正规口岸、越区、拍摄边防/军事管理区、或任何触碰 SPEC 既有红线（翻墙、非法越境）的行为。与签证分区标注同属合规要求。
